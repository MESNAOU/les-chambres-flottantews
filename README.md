# les-chambres-flottantews
Ce projet constitue une soumission pour un examen dans le cadre du module "Conception et programmation 2D/3D" que j'ai étudié cette année. Il s'agit d'un jeu dans lequel les joueurs doivent sauter d'une chambre flottante à une autre afin d'atteindre le drapeau de l'INPT. La vidéo ci-jointe présente une démonstration de l'exécution du jeu réalisée par mes soins.
